# Codex 作業時間記録

## 目的

Codex に依頼した作業の開始時間・終了時間・合計作業時間を記録するためのログです。

## 記録ルール

- 作業開始時に 1 行追加する
- 作業終了時に終了時間と合計作業時間を追記する
- 日付は `YYYY-MM-DD` 形式
- 時刻は `HH:MM` 形式で統一する
- 合計作業時間は `○時間○分` または `○分` で記録する
- 記録は `作業ログ` へ記録する
- 記録はによりNo.が更新される時は下へ追加記録していく
